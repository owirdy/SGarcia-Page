# Sofia Garcia - Academic Personal Website

This is the academic personal website of **Sofia Garcia**, a Ph.D. Candidate in Robotics at ETH Zurich.

Built using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template and hosted on GitHub Pages.

## About

Sofia Garcia is a third-year PhD candidate at the ETH Zurich Robotics Systems Lab, focusing on robot learning and control, including reinforcement learning for legged locomotion, sim-to-real transfer, and dexterous manipulation.

## Site Structure

- `_pages/about.md` - Homepage with bio and publications
- `_pages/cv.md` - Curriculum Vitae
- `_pages/publications.md` - Publications list
- `_publications/` - Individual publication pages
- `_config.yml` - Site configuration
- `_data/navigation.yml` - Navigation menu
